기본 검색

검색창에 키워드 또는 질문을 입력 후 검색 버튼 또는 Enter를 누릅니다.

예시: 감가상각 · 법인세율 · 퇴직연금 DB형 회계처리 · 이사 보수 주주총회 결의 없이 지급 가능한가요
🆕 자연어 질문 — 고품질 AI 종합답변

물음표로 끝나거나 "~인가요/하나요/왜/어떻게/얼마나" 같은 완전한 문장으로 질문하면, DB에 없는 새로운 질문이라도 Gemini AI가 소제목·단계별 설명·비교표까지 포함한 상세 답변을 만들어드립니다(예: "개념 설명 → 구체적 수치·사례 → 실무 팁" 순서로 구성). DB에 이미 등록된 근거자료가 있으면 그 카드도 함께 표시됩니다. 답변에 등장하는 각 법령 조문은 클릭하면 law.go.kr에서 해당 조문으로 바로 이동합니다.

예시: 최저임금 안 지키면 어떻게 되나요? · 퇴직할 때 세금은 얼마나 내나요

검색창 아래 체크박스로 검색할 출처를 선택합니다. 체크 해제 시 해당 출처 카드가 생성되지 않습니다. 분류별(📘회계 · 🏦세무 · ⚖️법률 · ⚡AI)로 정렬되어 있습니다.

  • 📘 KIFRS 기준서 · KIFRS 질의회신 · DART — 회계기준·공식해석·기업공시
  • 🏦 국세법령 · 국세청 · 조세심판 · 기재부 — 세법·예규·심판결정례·세제개편
  • ⚖️ 국가법령 · 전자소송 · 판례검색 · 고용부 — 법령·소송절차·판례·노동해석
  • ⚡ AI법률 — 빅케이스 AI 판례 검색 (모든 검색어에 자동 추가)
판례 번호 검색

사건번호를 입력하면 빅케이스 AI · 국세법령정보시스템 · 국가법령정보센터 3개 출처로 즉시 연결됩니다. 빅케이스 AI는 사건번호를 인식해 상세 판례 패널까지 자동으로 엽니다.

예시: 2016다264556 · 87누362 판결 · 2024두12345

바로가기 ↗ 버튼

클릭 시 해당 공식 출처 페이지로 직접 이동합니다. 국가법령정보센터(law.go.kr) 관련 카드는 기본적으로 해당 법령의 검색결과 페이지로 안정적으로 연결됩니다(2026.7 재검증 — 예전에는 조문번호를 임의로 추측해 링크를 만들다 보니 간혹 엉뚱한 화면으로 연결되는 문제가 있었는데, 이제는 항상 정상 작동하는 방식만 사용합니다). ⚙️ 조문 정밀 링크 프록시를 설정해두면(아래 활용 팁 참고) 검색결과 페이지를 거치지 않고 실제 조문 화면까지 한 번에 바로 열립니다.

상세보기

클릭 시 Gemini AI의 전문 분석·요약·근거 조문·관련 출처 링크를 모달로 확인합니다. 근거조문이 여러 개 나열된 경우("최저임금법 §6, §28" 등) 각 조문이 개별 링크(pill)로 따로 표시되어, 원하는 조문만 골라 바로 열 수 있습니다.

📎 파일·이미지 첨부 검색

사용 방법
  1. 검색창 내부 우측 📎 파일 업로드 버튼 클릭
  2. 파일 선택 (복수 선택 가능)
  3. 첨부 파일 미리보기 확인 · ✕로 제거 가능
  4. 검색어 입력(선택) 후 검색 클릭
  5. Gemini AI가 파일 내용을 분석하여 관련 출처 카드 생성
지원 파일 형식
  • 🖼 이미지 — PNG · JPG · GIF · WebP (재무제표 캡처, 계약서 사진 등, Gemini가 직접 인식)
  • 📄 PDF — 감사보고서, 계약서, 판결문 등 (Gemini가 직접 텍스트·이미지 인식)
  • 📝 Word — .docx만 지원 (mammoth.js로 텍스트 직접 추출)
  • 📊 Excel — .xlsx · .xls · .csv (시트별 표 내용을 텍스트로 변환 후 분석)
  • 📊 PowerPoint — .pptx만 지원 (슬라이드별 텍스트 추출)
  • 📃 텍스트 — .txt
  • ⚠️ 구버전 형식(.doc·.ppt)과 한글(.hwp)은 아직 미지원 — PDF로 변환해서 첨부해주세요
활용 예시
  • 재무제표 이미지 첨부 → 회계·세무 쟁점 자동 분석
  • 계약서 PDF 첨부 → 관련 법령·판례 출처 카드 생성
  • 판결문 이미지 첨부 → 관련 조문·유사 판례 연결
  • 검색어 없이 파일만 첨부해도 자동 분석

⚠️ 회계·세무·노동·법률 리스크 검증

무엇을 하는 기능인가요

상단 "⚠️ 리스크 검증" 탭에서, 재무제표·결산부속명세서·계정별원장·거래처원장·법인세 세무조정계산서·계약서·근로계약서·외주계약서·사내규정 등을 첨부하면 Gemini AI가 회계·세무·노동법 관점에서 확인이 필요한 지점을 찾아냅니다. 일반 검색(파일 첨부)이 "관련 출처를 찾아주는" 기능이라면, 이 기능은 "문제가 될 만한 부분을 미리 짚어주는" 진단 전용 기능입니다.

사용 방법
  1. 상단 "⚠️ 리스크 검증" 탭 클릭
  2. (선택) 상단 입력창에 특히 중점적으로 확인하고 싶은 사항을 문장으로 입력 — 예: "가지급금 인정이자 계산이 맞는지 집중적으로 봐주세요"
  3. (선택) 📎 파일 선택으로 검토할 자료 첨부 (여러 개 동시 첨부 가능)
  4. "🔍 리스크 검증 시작" 클릭 — 파일 없이 질문만으로도, 파일만으로도, 둘 다로도 검증 가능
  5. 심각도별(🔴높음 · 🟡주의 · 🟢참고)로 정리된 발견사항을 근거 법령·권장 조치와 함께 확인 — 근거 법령은 클릭하면 law.go.kr 해당 조문으로 바로 이동합니다(여러 조문이 있으면 각각 개별 링크로 분리)
진단 범위
  • 👷 노동 — 최저임금·주52시간 초과·근로계약서 필수기재사항(근로기준법 §17) 누락·포괄임금제 위법 소지·4대보험/퇴직금 미적립·위장도급(불법파견) 소지·취업규칙 신고의무 등
  • 🏦 세무 — 특수관계자 거래·접대비 한도초과·원천징수 누락·가지급금 인정이자·부가세 과세/면세 구분 오류·세무조정 누락 등
  • 📘 회계 — 재무제표·부속명세서상 정합성, 회계처리 쟁점 등
지원 파일 형식

일반 검색의 파일 첨부와 동일합니다 — PDF·이미지·엑셀(.xlsx/.xls)·워드(.docx)·파워포인트(.pptx)·txt/csv. 구버전(.doc/.ppt)·한글(.hwp)은 PDF로 변환해서 첨부해주세요.

🤖 AI 자동 분석

DB·KNOWLEDGE에 없는 검색어는 Gemini AI가 자동 분석하여 출처별 카드를 생성합니다. M 아이콘 애니메이션이 표시되는 동안 분석 중이며, 완료 시 카드로 자동 교체됩니다.

⚡ AI법률 체크박스

검색 범위에서 AI법률을 체크하면 모든 검색어에 빅케이스 AI 판례 카드가 자동 추가됩니다. 해제 시 판례 카드가 생성되지 않습니다.

📌 개정 동향 더보기

사이드바 개정 동향에서 더보기 클릭 시 회계기준(5개+)·세무(16개+)·법률(9개+) 분류별 전체 개정사항을 확인할 수 있습니다. 항목 클릭 시 바로 검색됩니다.

📱 모바일 사용

모바일에서는 분야별 검색이 가로 스크롤 칩으로 압축되고, 검색 히스토리 패널은 숨김 처리됩니다. 파일 첨부(📎)도 모바일에서 동일하게 지원됩니다.

바로가기·근거 링크 클릭 시, 서버가 law.go.kr Open API로 실제 시행 중인 조문을 조회해 정확한 조문 화면까지 바로 이동합니다(법 개정으로 조문이 바뀌어도 항상 최신 반영). 별도 설정이 필요 없으며, 서버 응답이 없거나 실패해도 3초 이내에 자동으로 법령 검색결과 링크로 전환되어 깨진 링크가 뜨는 일은 없습니다.
